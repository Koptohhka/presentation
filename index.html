Interfaces in the TypeScript

An interface is a programming structure/syntax that allows the computer to enforce certain properties on an object (class). For example, say we have a car class and a scooter class and a truck class. Each of these three classes should have a start_engine() action. 

					
    //Keyword "interface"
	interface currentWeatherInterface {
		currentTemperature: number,
		currentWeatherDescription: string,
		getCurrentTemperature: () => number,
		moreInfo: {
			humidity: number,
			windSpeed: string;
		} 
	}

	//following object
	const todayWeatherObj:currentWeatherInterface = {
		currentTemperature: 17,
		currentWeatherDescription: 'cloudy',
		getCurrentTemperature: () => {
			return this.currentTemperature;
		},
		moreInfo: {
			humidity: 83,
			windSpeed: '2 m/s'
		}
	}
